# Tugas-Besar-IF4806-Kelompok_16
Rancanglah multi-linked list yang memodelkan bis, penumpang, dan hubungan di antara mereka. Sebuah bis dapat dinaiki lebih dari satu penumpang, dan satu penumpang hanya dapat naik satu bis. Program ditujukan untuk mengelola trayek bis berdasarkan rute dan kapasitas penumpang.
